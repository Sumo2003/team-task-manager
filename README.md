# 📝 Team Task Manager

A full-stack Team Task Manager web application that allows users to manage projects, assign tasks, and track progress efficiently.

---

## 🚀 Features

- 📋 Create and manage projects  
- ✅ Assign and track tasks (To-do, In-progress, Done)  
- 🔐 User authentication (Login/Register)  
- 📊 Dashboard for project overview  
- ⚡ Real-time API integration  
- 🎯 Clean and responsive UI  

---

## 💡 Motivation

This project is built to simplify team collaboration and task tracking.  
It helps users organize their work, monitor progress, and improve productivity.

---
